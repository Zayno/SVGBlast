# SVG Blast

## SVG Blast is a lightweight Windows app to view SVG files

![SVGBlast Screenshot](/SVGBlastProject/Screenshots/SVGBlast_Screenshot.png "This is a sample image.")

### Features:

* Very lightweight and fast (200KB).
* Can ZOOM, PAN and RESET.
* Can be used as the Default app for SVG Files.

### Installing:

* Download SVGBlast.exe.
* Place at a desired directory.
* In Windows Settings -> Default Apps, Set SVGBlast as .SVG default APP.

  
### Usage:

* Once set as the default app for .SVG files, double click on any SVG file to view with SVGBlast.
* Alternatively, you can use the Command Line.
* Use Left Mouse button to Pan.
* Use Mouse Wheel to Zoom.
* Use Right mouse button to Reset the View and Zoom.
* Use Space key to Sharpen and regenerate the SVG.
