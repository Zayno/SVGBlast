# SVG Blast

## SVG Blast is a lightweight Windows app to view SVG files

![SVGBlast Screenshot](/SVGBlastProject/Screenshots/SVGBlast_Screenshot.png "This is a sample image.")

### Features:

* Very lightweight and fast (200KB).
* Can ZOOM, PAN amd RESET.
* Can be used as Default app for SVG Files.

### Usage:

* Download SVGBlast.exe.
* Place at a desired directory.
* In Windows Settings -> Default Apps, Set SVGBlast as .SVG default APP.
